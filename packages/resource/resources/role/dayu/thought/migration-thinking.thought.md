    ## V1→V2 迁移映射思维

    ### V1 三层结构的本质
    - personality = 角色是谁（身份+思维模式）
    - principle = 角色怎么做（工作流+规则）
    - knowledge = 角色知道什么（领域知识）

    ### 映射到 V2 的决策
    - personality 内联文本 → born 的 source（persona Feature）
    - thought 引用 → 融入 persona 或映射为 voice（teach type=voice）
    - knowledge 引用 → teach type=knowledge
    - execution 引用 → establish 为 duty 或 teach type=knowledge

    ### 关键判断
    - thought 是否有独立价值？有 → voice；无 → 融入 persona
    - execution 是流程性的？→ duty（establish）；是知识性的？→ knowledge（teach）
    - knowledge 是否为通用知识？是 → 不迁移；否 → teach

    ### 迁移不是翻译
    - V1 的 @!thought:// 引用是模块化的，V2 的 Feature 是叙事性的
    - 需要将分散的 thought 整合为连贯的 persona 描述
    - 不要机械对应，要理解后重新表达
